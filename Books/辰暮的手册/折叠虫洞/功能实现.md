# 折叠虫洞功能实现

## 1. 技能系统架构

### 1.1 设计原则

- **数据驱动**: 技能模板通过 JSON 配置定义
- **行为段组合**: 技能由多个 Segment 组成

### 1.2 核心类

| 类名 | 职责 |
|------|------|
| `AbilityContext` | 技能上下文，生命周期数据传递 |
| `AbilitySegment` | 行为段基类 |
| `AbilityTemplate` | 技能模板 |
| `CasterAbilityComponent` | 施法者组件 |

### 1.3 已实现行为段

| Segment | 功能 |
|---------|------|
| `MoveSegment` | 位移行为段，支持瞬移/平滑移动 |
| `ThrowSegment` | 投射物行为段 |

### 1.4 配置参考

- 技能模板定义: [Ability模块架构文档.md](技能系统/Ability模块架构文档.md)
