# 折叠虫洞策划案

## 1. 核心玩法

### 1.1 六边形网格地图

- 使用 HexGrid + Tile 系统管理地图格子
- 玩家在六边形网格上进行移动和战斗

### 1.2 虚数核心 (ICore)

游戏核心实体，具有以下状态：

| 状态 | 描述 |
|------|------|
| 脱离态 | ICore 独立存在，可自由移动 |
| 伴随态 | ICore 跟随玩家 |
| 刃嵌入 | ICore 嵌入刃武器 |
| 爪嵌入 | ICore 嵌入爪武器 |
| 枪嵌入 | ICore 嵌入枪武器 |

## 2. 技能设计

### 2.1 基础技能

三个基础技能：

- **人物能力**: 移动
  - 未蓄力形态：**移动 (move)** - 距离1
- **武器1能力**: 投掷
  - 未蓄力形态：**飞刀 (knife)** - 距离2
  - 蓄力形态：**投掷 (throw)** - 距离2
- **武器2能力**: 近距离施法
  - 未蓄力形态：**爪击 (claw)** - 距离1
  - 蓄力形态：**虫洞 (wormhole)** - 距离1

### 2.2 技能形态

每个技能具有两种形态：

- **蓄力形态**: 长按蓄力后释放更强力
- **未蓄力形态**: 点击即释放

### 2.3 技能按钮功能

技能操作方式如下：

1. **蓄力**: 通过 ChargeButton 组件实现长按蓄力
2. **方向指向**: 通过 ArrowButton 组件实现拖拽指向6个方向
3. **技能释放**: 拖拽并放开时视为释放技能（若有指向方向）
4. **取消释放**: 无指向方向时松开视为取消释放技能
5. **移动技能**: 无蓄力功能，无指向功能，方向由虚拟摇杆UI决定，点击即可向所指方向移动一个单位

### 2.4 技能配置

- 技能模板定义文件: `Assets/AssetRaw/Configs/AbilityTemplates.json`

## 3. 坐标系统

### 3.1 坐标系选择

使用斜角坐标系 (Oblique) 处理六边形网格

### 3.2 坐标转换

CoordLib 工具类提供以下转换：

- `HexToRect`: 六边形坐标转矩形坐标
- `RectToHex`: 矩形坐标转六边形坐标
- `HexToWorld`: 六边形坐标转世界坐标
- `WorldToHex`: 世界坐标转六边形坐标
