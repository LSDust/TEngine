# 折叠虫洞概况

## 1. 架构与技术栈

采用TEngine开源框架 可调用tengine-dev Skill: [SKILL.md](../../.trae/skills/tengine-dev/SKILL.md)

技术栈: HybridCLR + YooAsset + UniTask + Luban

## 2. 适用平台与输入方案

**平台**: 移动端

**输入方案**: Input System [输入方案架构文档.md](输入方案/输入方案架构文档.md)

## 3. 项目类型

单机项目

## 4. 项目运行流程与生命周期

- ProcedureSplash → 启动画面
- ProcedureLaunch → 框架初始化
- ProcedurePreload → 预加载
- ProcedureMenu → 菜单
- ProcedureBattle → 战斗

## 5. 依赖资源的储存方式

- 技能配置: `Assets/AssetRaw/Configs/AbilityTemplates.json`
- UI预制体: `Assets/AssetRaw/UI/`
- 场景: `Assets/AssetRaw/Scenes/`
